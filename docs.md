Commands:
```
!ping                          |say hi
!currency <a> <b> <amount?>    |convert currency
!cc <a> <b> <amount?>          |convert currency
!<a><b> <amount?>              |convert currency
!=<math expression>            |run a math expression
!uuid                          |generate a uuid
!rng <type=i> <a?> <b?>        |random number between A and B (inclusive)
                                type = `i`, `int`, `f`, `float`
!cap <text>                    |capitalizes text
!low <text>                    |lowercases text
!mock <text>                   |"mocks" text (lowercase uppercase mix).
!todo <text>                   |adds a thing to your online text storage
                                this text can be retrieved at [WIP] using
                                your secret key that is obtained with !key.
!key                           |gets your key.
!rl                            |<dave only> [WIP] reloads and updates bot
```
